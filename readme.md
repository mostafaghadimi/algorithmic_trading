# Asset Classes

</div>
 کلاس دارایی انواع مختلفی از وسایل/ابزارهای واقعی است که برای تبادل‌های مختلف در دسترس است.

 **future**: قراردادی است که بین خریدار و فروشنده بسته می‌شود و در آن تضمین می‌شود تا یک تاریخ مشخص خریدار، یک مقدار مشخص از کالای تهیه شده توسط خریدار را با قیمت تعیین شده خریداری کند.
